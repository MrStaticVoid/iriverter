3.3. Advanced Functions

These functions were designed for advanced users only. If you don't know what you are doing, you could create unplayable videos or crash iriverter

3.3.1. Manually Splitting Videos

iriverter can automatically split videos to a specified length for easy seeking in long videos. For the H300, it is necessary for videos to be under an hour in length. It tries to make equally sized parts, but if you want to take control of the splitting yourself unselect Options->Advanced->Automatically Split and convert your videos as you would normally. When that is done select Jobs->New->Advanced->Manual Split, choose your input video, add the times to split around and press Convert again. You will get a video file for each desired split in the same directory as the input video.

To determine where to put the split marks, you can open the video file in iriverter's video player by selecting Jobs->Play File... (Shift-Ctrl-F). You can see the current position by pressing O to activate the on-screen-display. You can move forward and backwards by pressing Right or Left, respectively. You can pause by pressing P.

3.3.2. Joining Videos

iriverter can merge any number of videos into one. This is handled as a conversion job and can be started from Jobs->New->Advanced->Join Videos. Continue to add the videos that you want to merge, select a place for the output file to be written and start the conversion as outlined in Section 3.1.4.

iriverter does not impose any restrictions on the videos that you can merge, however, if the output is to be playable, the input videos must be of the same video and audio formats, resolution, and bitrates.

3.3.3. Fixing Audio/Video Sync

If you find that the audio is out of sync with the video, you can specify an amount of time up to 1000 milliseconds with which to delay or advance the audio stream in Options->Advanced->Audio Sync....

If you are using Linux and do not have the MPlayer patch applied, you cannot bring the audio ahead of the video. See Section 2.3 for more information.

3.3.4. Profiles

iriverter has limited conversion profiling support. If you have a new video-playing device, it is possible to write a profile for iriverter to convert videos for it. Profiles can be found in the profiles directory in iriverter's data directory, usually /usr/share/iriverter or /usr/local/share/iriverter if you are using Linux. A simple profile looks like the following and should be self-explanitory:

Example 3-1. pmp.profile

device=PMP Series
maxVideoBitrate=1500
maxAudioBitrate=192
dimensions=320x240 512x384 640x480
maxFrameRate=30
