Chapter 2. Installing iriverter

Before beginning, fetch the latest version of iriverter for your platform from iriverter's website.

2.1. Windows

There are both installers and ZIP packages of iriverter available for Windows. Both will give the same result in the end, but using the installer is recommended.

2.1.1. Installer

  1. Double click iriverter-VER_win.exe, where VER is the version number.

  2. Follow the on-screen instructions

  3. Start iriverter with Start->Program Files->iriverter->iriverter.

2.1.2. ZIP

  1. Extract iriverter-VER_win.zip, where VER is the version number, using a ZIP extraction program like WinRAR or WinZIP. Windows XP includes an unzipper by default so the ZIP will open like a folder where you only have to copy and paste the contents out.

  2. Find where you extracted the contents of the ZIP and double-click iriverter.exe to start iriverter.

If you want to move iriverter to another place, move the whole contents of the iriverter folder around together. Do not move iriverter.exe around by itself. If you want to be able to excute it from another place, make a shortcut to iriverter.exe
